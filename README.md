### 一、项目概述

1. 目标：理解前后端，具备独立搭建前后端分离项目的能力。
2. 开发模式：前端+后端+数据库。
3. 技术栈：
    1. 前端：vue+vuex+ElementUI+Axios+vue-element-admin说明：前端框架、全局状态管理框架、前端UI框架、前端HTTP框架、项目脚手架。
    2. 后端：springBoot+Mybatis+Mybatis-plus+Redis 说明：容器+MVC框架、ORM框架、MyBatis增强工具、非关系型数据库。
